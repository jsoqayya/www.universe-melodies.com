🚀 طريقة الرفع:
───────────────────────────────────────────────────────────────

1. افتح: https://play.google.com/console
2. اختر تطبيقك: "ترانيم الكون"
3. اذهب إلى: Store presence → Main store listing

رفع الأيقونة:
- قسم "App icon"
- اضغط Upload
- ارفع: app_icon_512x512.png

رفع البانر:
- قسم "Feature graphic"
- اضغط Upload
- ارفع: feature_graphic_1024x500.png

رفع لقطات الهاتف:
- قسم "Phone screenshots"
- اضغط Add screenshots
- ارفع الـ 5 صور بالترتيب:
  1. screenshot_1 (الغلاف العربي)
  2. screenshot_2 (English Version)
  3. screenshot_3 (قراءة سلسة)
  4. screenshot_4 (محتوى شامل)
  5. screenshot_5 (بدون إنترنت)

رفع لقطة التابلت (اختياري):
- قسم "7-inch tablet" أو "10-inch tablet"
- اضغط Add screenshots
- ارفع: tablet_screenshot_1.png

4. اضغط Save
